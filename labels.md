# Required Labels

> #### Priority
| Label | Color Name | Hex Code | Use For |
|---|---|---|---|
| `priority::critical` | Crimson Red | `#d73a4a` | Must fix immediately, do this first |
| `priority::high` | Safety Orange | `#ff6b35` | Important, do soon |
| `priority::medium` | Orange | `#ffa500` | Normal priority |
| `priority::low` | Light Gray | `#d3d3d3` | Can wait |

---
> #### Status
| Label | Color Name | Hex Code | Use For |
|---|---|---|---|
| `status::blocked` | Crimson Red | `#d73a4a` | Blocked, cannot move forward |
| `status::in-progress` | Golden Yellow | `#ffb819` | Someone is working on it |
| `status::needs-review`| Ocean Blue | `#0075ca` | Waiting for review |
| `status::ready` | Success Green | `#28a745` | Ready to merge/deploy |

---
> #### Type
| Label | Color Name | Hex Code | Use For |
|---|---|---|---|
| `type::bug` | Crimson Red | `#d73a4a` | Something is broken |
| `type::docs` | Ocean Blue | `#0075ca` | Docs need updating |
